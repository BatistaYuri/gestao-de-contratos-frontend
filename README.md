# Gestão de Contratos — Frontend

## Requisitos

- Node.js (versão utilizada no desenvolvimento: `v24.15.0`)
- API do projeto em execução

## Como executar

1. Abra um terminal na pasta do projeto.

2. Crie um arquivo chamado `.env` na raiz do projeto:

```env
VITE_API_URL=http://localhost:3001
```

Altere a URL caso a API esteja rodando em outro endereço.

3. Instale as dependências:

```bash
npm install
```

4. Inicie o projeto:

```bash
npm run dev
```

5. Abra no navegador o endereço exibido no terminal, normalmente:

```text
http://localhost:5173
```
